# 数字电路复习

一个用静态网页整理数字电路课程内容的复习页面，包含目录导航、章节内容、表格和配图说明。

## 功能

- 首页目录导航
- 章节跳转与返回目录
- 按知识点分块展示内容
- 表格展示编码、真值表等信息
- 插入课程相关截图辅助说明
- 基础移动端适配

## 技术栈

- HTML
- CSS
- 原生 JavaScript

## 开发环境

- 现代浏览器

## 项目结构

- 主页面：`数字电路/数字电路复习.html`
- 配套图片：位于 `数字电路/` 目录下，并在页面中作为辅助说明素材使用

## 页面内容

页面内容覆盖以下模块：

- 数制与进制转换
- 二进制算术运算
- 原码、反码、补码
- 常用编码与格雷码
- 逻辑代数基本运算
- 逻辑代数公式与定理
- 逻辑函数表示方法
- 逻辑函数化简

## 核心实现

### 页面组织

页面采用目录页和章节页结构。首页使用卡片列出主要复习模块，点击后切换到对应章节。

### 内容展示

页面中组合使用了：

- 卡片目录
- 提示块
- 表格
- 公式区块
- 图片说明区

### JavaScript 交互

脚本负责页面切换、隐藏与显示当前章节，并在切换后滚动回顶部。

## 如何运行

这是一个静态页面项目，直接用浏览器打开即可：

```text
数字电路/数字电路复习.html
```

## 项目展示

项目目录中包含多张配套截图，并在页面中引用。打开主页面即可查看完整效果。
